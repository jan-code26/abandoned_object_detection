About the project

This website deals with the detection of Abandoned Object. This paper is focused towards developing a computer vision-based approach that analyzes the blob areas to detect any abandoned objects and instantaneously send appropriate alerts without any human intervention. Documentation of Project

Upon user video upload, the following steps are taken for abandoned object detection

  • Initialization: A static picture is captured from the uploaded video, which serves as a reference frame. This initial frame should not contain any suspicious or moving objects. It provides a baseline for comparison.
  • Frame Comparison: Each subsequent frame of the video is compared to the initial frame. The difference between the static picture and the real-time frames is computed to identify any changes or discrepancies.
  • Object Identification: The filtered differences are analyzed to identify and track objects. The position and state of each object are continuously monitored throughout the video.
  • Abandoned Object Detection: If an object is detected to be in motion, no further action is taken. However, if an object remains stationary in the same position for a certain duration, it is considered as an abandoned or suspicious object. This signifies a potential security concern that requires attention.
